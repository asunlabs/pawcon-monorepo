# PawCon Server Side

## Architecture
|Type|Before|After|
|:-----|:----|:---|
|Front End|HTML, CSS, JS|HTML5, CSS3/Sass, React, JS/TS|
|Backend|X|Node.js, Express|
|Database|X|MongoDB|
|Test|X|Mocha, Jest|
